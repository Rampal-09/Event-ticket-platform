# System Roles

The system includes three main roles.

---

# 1. Attendee (Public User)

Attendees are users who browse events and purchase tickets.

Permissions:

* View events
* View event details
* Purchase tickets
* Receive QR code tickets

Restrictions:

* Cannot create events
* Cannot scan tickets

---

# 2. Organizer

Organizers manage events.

Permissions:

* Access organizer dashboard
* Create events
* Edit events
* View ticket sales
* Manage their own events

Restrictions:

* Cannot scan tickets
* Cannot access other organizers' events

---

# 3. Entry Staff

Entry staff manage event gate entry.

Permissions:

* Access QR scanner page
* Scan ticket QR codes
* Validate ticket status
* Mark ticket as used

Restrictions:

* Cannot create events
* Cannot manage tickets
* Cannot access organizer dashboard

---

# Role Access Summary

Attendee

* Browse events
* Purchase tickets

Organizer

* Create and manage events
* Track ticket sales

Entry Staff

* Scan tickets
* Validate event entry
